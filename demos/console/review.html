Browser Developer Tools for Non-Developers

The Console allows you to:

  1. Try out JavaScript.
  2. Read and change content of the document in the browser programmatically.

Benefits:

  1. Use JavaScript to get access to anything in the document or the browser.

Issues:

  1. Information overload - you need to filter results.
  2. Users can be duped into writing code that leaks their information to he web.
