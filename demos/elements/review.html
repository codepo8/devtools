Browser Developer Tools for Non-Developers

The Elements tool allows you to:

  1. See the structure of the current web page and navigate its elements.
  2. Change and delete parts of the page.
  3. Change the style and look of the current web page.

Benefits:

  1. Acesss to all parts of the document.
  2. Quick navigation using your keyboard.

Issues:

  1. Lots of web products these days have complex structures and are hard to change.
