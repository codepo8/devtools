Browser Developer Tools for Non-Developers

The device emulation tool allows you to:

  1. Access simpler mobile optimised versions of the current page.
  2. Change the width of the current document without resizing the browser.
  3. Check the dimensions of page elements using rulers.
  4. Take screenshots of whole documents.
  5. Simulate mobile connection speeds.
  6. See the current page in different modes without having to change your OS.

Benefits:

  1. Use the mobile version of web sites on your computer.
  2. Check how your product performs in different environments.

Issues:

  1. Devices are only simulated by size, a mobile will still show products differently.
  2. You work in an emulated touch mode - mouse dependent functionality can not be used.
